#### 用户表

| 字段     | 类型   | 大小 | 描述            |
| -------- | ------ | ---- | --------------- |
| username | string | 255  | 用户名唯一      |
| id       | string | 255  | ID主键          |
| password | string | 255  | 密码            |
| 角色     | string | 255  | 身份 默认为博主 |
| age      | string | 255  | 年龄            |
| sex      | string | 255  | 性别            |
| imgurl   | string | 255  | 头像地址        |
